Resources
One primary source per topic. Finish it. Move on. No course hopping.
⚠️
The anti-pattern that kills progress Collecting 10 courses and finishing none. Every resource here is curated for your specific plan. If the primary isn't clicking after 2 days, switch to the backup — not a third option you find on Reddit.
Your enrolled courses — use these first
C# · Weeks 1–3 TCS Udemy: C# Beginners
Your enrolled course — use for Weeks 1–3. Language core, OOP, LINQ. Stop when you're comfortable writing a class with interfaces and chaining LINQ. Don't finish the whole thing if it starts covering topics ahead of schedule.
ASP.NET · Weeks 4–6 TCS Udemy: Build REST APIs
Your enrolled course — primary source for Weeks 4–6 and Project 1. Covers controllers, EF Core, Repository Pattern, AutoMapper, JWT in one flow. Type every line from scratch, never copy-paste. This course alone gets you to Project 1.
Free resources — backend & architecture
Architecture · Wks 7–8 Milan Jovanović — YouTube free
youtube.com/@MilanJovanovic
Best free resource for modern .NET — clean architecture, CQRS, MediatR, vertical slice, domain events. This channel alone covers Phase 3 of the .NET section. Watch, pause, implement on Project 1 in parallel.
Testing + advanced · Wks 9–13 Nick Chapsas — YouTube free
youtube.com/@nickchapsas
Search: "unit testing .NET", "integration testing ASP.NET Core", "JWT refresh tokens .NET", "rate limiting .NET 7". Covers everything from Phase 4–5. His content is production-grade — not beginner-padded.
ASP.NET Backup Kudvenkat — YouTube free
youtube.com/@kudvenkat
If the Udemy course isn't clicking, switch here. Very detailed, methodical pace. Search "ASP.NET Core Web API kudvenkat". Indian instructor, excellent for building intuition step by step.
Clean arch reference Jason Taylor Template — GitHub free
github.com/jasontaylordev/CleanArchitecture
Don't clone and use blindly. Open the repo, read every folder, understand why each file is where it is. This is the real-world reference implementation. Use it as a map, not a starting point.
Azure
Azure · Wks 10–16 MS Learn — AZ-204 Path free + sandbox
learn.microsoft.com → AZ-204
Don't study for the exam. Use the learning modules — they include free Azure sandbox environments. App Service, SQL, Blob, Service Bus, Key Vault, App Insights are all covered with hands-on labs. Do the lab, then apply to your project.
CI/CD GitHub Actions Docs free
GitHub Actions → Deploy .NET to Azure
The official quickstart for deploying .NET to Azure App Service via GitHub Actions is exactly what you need. Follow it verbatim for Project 1's pipeline. It covers build, test, and deploy steps.
React
React React Official Docs free
react.dev/learn
The new docs (react.dev) are genuinely the best React learning resource. Start with "Quick Start" and "Thinking in React". These two pages alone give you the mental model. Do the interactive examples, don't just read.
JavaScript The Modern JS Tutorial free
javascript.info
The best free JavaScript resource online. Read Part 1 (Language) — specifically: functions, objects, arrays, promises, and async/await. Skip the browser and DOM sections for now; focus on the language itself.
Interview preparation
DSA · Sep–Oct Neetcode — Blind 75 free
neetcode.io/practice
Don't do all 75. Focus on: Arrays, Strings, Hash Maps, Two Pointers, Sliding Window. That's ~25 problems and covers 80% of what backend .NET roles ask. Solve in C# — not Python — so you practice the language you'll interview in.
System Design · Sep System Design Primer free
github.com/donnemartin/system-design-primer
Read the "How to approach" section and the basics: load balancers, caching, databases, queues, CDN. You don't need deep system design for 2YOE roles — just enough to talk intelligently and not blank out.
Optional paid Dometrain — Clean Architecture ~$50
dometrain.com
Made by Nick Chapsas. Worth it if you have the budget, but Milan's free YouTube channel covers 80% of the content. Only buy this if you want a structured course experience for clean architecture.
ASP.NET reference MS Docs — ASP.NET Core free
learn.microsoft.com/aspnet/core
Don't use as primary learning. Use as reference when you're stuck on something specific — middleware, auth, EF Core. The docs are excellent for looking up "how does X work exactly" after you've already learned the concept.
